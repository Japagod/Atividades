O JavaScript é uma linguagem de programação criada pela Netscape em 4 de dezembro de 1995 e é utilizado para criar páginas Web interativas e dinâmicas. Há três formas de adicionar o JS em um site: Usando a tag Style dentro da tag head (Usado para itens que precisam ser carregados antes da página), Usando a tag Style dentro da tag Body (usado para itens que não precisam carregar antes da página) e/ou em um arquivo Style.js (usado para organizar todo suas linhas em um único documento, necessário devido ao tamanho que alguns projetos ocupam). O JS tem várias versões, como o ES5, ES6 e ES2015, sendo a ES2018 a mais atual. Mas nem sempre precisamos criar todo o nosso código, para isso usamos bibliotecas, elas são como blocos de lego que podemos implementar no nosso site podendo também mudar “a composição” de cada bloquinho. Para adicionar uma biblioteca, é usado o parâmetro src com o link da biblioteca. Aqui vai alguns exemplos de bibliotecas:

KendoUI: como o nome sugere, a biblioteca ajuda na criação de interfaces

Quill: adiciona campos de textos completos, tendo até mesmo como enviar fotos vídeos e etc…

FullCalendar: outra biblioteca com nome autoexplicativo, com ele é possível adicionar calendários com várias funções interessantes.

ToastUI: esta por sua vez permite adicionar uma ferramenta de edição de imagem no seu site

Sweetalert: substitui o alert convencional do browser por uma versão estilizada
