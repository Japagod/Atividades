Atividade 1

  • O que é JavaScript? Quando foi criado?
  • Para que ele é usado em uma página HTML?
  • Como adicionar JS em uma página web?
  • O que é ES5 e ES6, ES2015, etc?
  • Qual a versão atual do JS?
  • O que é uma biblioteca JS e como adicionar uma ao site?
  • Cite 5 exemplos de bibliotecas (com suas funcionalidades principais).

Atividade 2

    Utilize o editor de sua preferência para criar sites com:

  • Texto que pode ter seu tamanho alterado por dois botões no início da tela.
  • Texto que pode mudar de cor de acordo com um botão de seleção de cor.
  • Número que pode ser ajustado por dois botões.
  • Permita que o usuário digite os valores de A, B e C e utilize-os para calcular as raízes da equação quadrática: y=Ax²+Bx+C.

Atividade Aula 3

    Crie uma calculadora de preços onde:

  • Cada maçã custa 1 real
  • Cada banana custa 0,5 real
  • Cada abacaxi custa 1,5 real
  • Compras acima de R$5 tem um desconto de 10%, acima de R$15 desconto de 15%
